GUIの起動：pixi run gui

project_pain/training_phase_spec.yamlに全ての実験条件が記述してある

tasks\painに本番実験に使うタスクがある

Develop a new task (manual_reward) in tasks\pain. This task assumes that @painTask_channels.py is loaded. In this task, when a button in the GUI is clicked, GateLED is on for 1 sec, then RewardValve is on for 0.05 sec. Also, cound and display the number of button click in the GUI. See https://pycontrol.readthedocs.io/en/latest/user-guide/programming-tasks/ for coding conventions in pyControl.
